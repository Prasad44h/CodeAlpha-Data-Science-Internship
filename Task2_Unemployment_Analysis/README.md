# Unemployment Analysis with Python

## Project Overview

This project analyzes unemployment trends in India using Python. It provides interactive visualizations and insights based on unemployment data, helping users understand employment patterns across different states and time periods.

## Features

- Interactive Dashboard
- State-wise Unemployment Analysis
- COVID-19 Impact Analysis
- Correlation Heatmap
- Employment vs Unemployment Visualization
- Key Insights and Recommendations

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Streamlit

## Dataset

**Dataset File:** `Unemployment.csv`

## How to Run

### 1. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn plotly scikit-learn streamlit
```

### 2. Navigate to the project folder

Open the terminal in the project folder.

### 3. Run the application

```bash
python -m streamlit run unemployment_analysis_dashboard.py
```
