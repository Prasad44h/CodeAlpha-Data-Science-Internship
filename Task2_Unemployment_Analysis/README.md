# Unemployment Analysis with Python

## Project Overview
This project analyzes unemployment trends in India using Python. It focuses on data cleaning, visualization, and exploratory data analysis to identify unemployment patterns across different states and time periods.

## Features
- Data Cleaning and Preprocessing
- State-wise Unemployment Analysis
- COVID-19 Impact Analysis
- Interactive Data Visualizations
- Correlation Analysis
- Key Insights and Findings

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Streamlit

## Dataset
Unemployment in India Dataset (CSV)
