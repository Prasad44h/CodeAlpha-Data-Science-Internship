# Car Price Prediction Dashboard

## Project Overview

This project predicts the prices of used cars using Machine Learning. It includes an interactive Streamlit dashboard where users can enter car details such as brand, model, year, fuel type, mileage, and transmission to estimate the car's selling price.

## Features

- Interactive Streamlit Dashboard
- Used Car Price Prediction
- Data Cleaning and Preprocessing
- Feature Engineering
- Machine Learning Model Training
- Real-time Price Estimation
- User-friendly Interface

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn
