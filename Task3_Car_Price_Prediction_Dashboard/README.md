# Car Price Prediction Dashboard

## Project Overview

This project predicts the prices of used cars using Machine Learning. It includes an interactive Streamlit dashboard where users can enter car details such as brand, year, fuel type, transmission, mileage, and other features to estimate the selling price of a used car.

## Features

- Interactive Streamlit Dashboard
- Used Car Price Prediction
- Data Cleaning and Preprocessing
- Feature Engineering
- Machine Learning Model Training
- Real-time Price Estimation
- User-friendly Interface

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn
- Plotly

## Dataset

**Dataset File:** `car data.csv`

## How to Run

### 1. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn plotly scikit-learn streamlit
```

### 2. Navigate to the project folder

Open the terminal in the project folder.

### 3. Run the application

```bash
python -m streamlit run car_prediction.py
```
